# Super Petz – E-Commerce Web Application

Super Petz is a full-stack e-commerce platform for pet products.  
Users can browse products, create accounts, add items to cart, and manage orders.

## Features

- User authentication (Login / Signup)
- Product listing and product details
- Shopping cart functionality
- Responsive UI for better user experience
- REST APIs for product and user management

## Tech Stack

Frontend
- React
- HTML
- CSS

Backend
- Node.js
- Express.js

Database
- MySQL

Deployment
- Godaddy Server
